# ElectroNagruzka

Проект на PlatformIO для создания электронной нагрузки на базе ESP32.

## Описание

Этот проект реализует электронную нагрузку с веб-интерфейсом для тестирования источников питания.

## Функции

- Калибровка нагрузки
- Режим разряда
- Веб-сервер для управления

## Структура проекта

- `src/main.ino` - основной файл
- `src/calib.ino` - калибровка
- `src/discharge.ino` - разряд
- `src/web.ino` - веб-интерфейс
- `platformio.ini` - конфигурация PlatformIO

## Установка

1. Установите PlatformIO
2. Склонируйте репозиторий
3. Откройте в VS Code с PlatformIO
4. Соберите и загрузите на устройство

## Использование

Подключите к источнику питания и управляйте через веб-интерфейс.
